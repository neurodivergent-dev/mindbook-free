# Changelog

All notable changes to the Mindbook Pro project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [4.0.1] - 2025-01-04

### Added

- **Offline Control**: Enhanced offline mode functionality with automatic synchronization when connectivity is restored
- **AI Support in Local Development**: Added local AI assistant support for developers with Microsoft Phi-1.5 model
- **Comprehensive Code Comments and Documentation**: Improved code readability with detailed comments and expanded documentation
- **Jest Tests**: Implemented comprehensive testing suite with Jest and React Native Testing Library

### Changed

- Optimized application performance for lower-end devices
- Improved UI responsiveness in tablet mode
- Enhanced encryption algorithm for vault notes

### Fixed

- Resolved synchronization issues when moving between WiFi and mobile data
- Fixed text formatting issues in Markdown editor
- Corrected localization errors in several languages

## [4.0.0] - 2025-20-03

### Added

- Initial public release of Mindbook Pro
- Multilingual support for 12 languages
- Secure note storage with Supabase
- Markdown-based rich text editor
- Theme customization options
- Cloud backup and restore functionality
- Category management system
- Advanced search capabilities
- Biometric security integration
- Vault system for encrypted notes
- Stylus and Apple Pencil support
- Real-time synchronization
